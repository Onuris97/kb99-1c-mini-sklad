# КБ99 Мини-Склад

Репозиторий проекта "Мини-Склад", созданного в рамках курса "1С Старт" компании КБ99

## Инструкция
Конфигурация выгружена в виде файлов .xml и .bsl. 

Чтобы загрузить конфигурацию в 1С:Предприятие, следуйте следующей инструкции:
1. [Скачайте](https://github.com/Onuris97/kb99-1c-mini-sklad/archive/refs/heads/main.zip) и распакуйте архив в любое удобное место,
2. Создайте новую информационную базу **без конфигурации**,
3. Запустите 1С:Предприятие в режиме «Конфигуратор»,
4. В меню конфигуратора выберите пункт «Конфигурация», а затем нажмите «Загрузить конфигурацию из файл**ов**»,
5. В пункте «Загрузить из» укажите «XML-файлы»,
6. Укажите путь к загруженной ранее папке kb99-1c-mini-sklad-main,
7. Ура! Всё готово!

## Примечание
- Версия 1С:Предприятия: 8.3.27.1936
